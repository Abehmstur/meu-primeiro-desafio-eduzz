Windows
- cd (navegar entre as pastas) ex: cd / (vai pra o local C) ex: cd windows
- dir (lista pastas)
- mkdir (mkdir cria pasta)
- del (deleta somente arquivos dentro da pasta)/rmdir 
- cls (limpar a prompt)
- Botão TAB (auto completa a palavra)
- echo (repete oque vc escreveu depois) 
- echo hello > hello.txt (senão houver um arquivo com este nome, ele irá cria-lo)
resultado: cria um arquivo hello.txt
